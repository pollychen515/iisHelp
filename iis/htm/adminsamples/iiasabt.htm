关于管理脚本

本主题包含关于管理脚本（包括 IIS Admin 对象、Cscript.exe 命令行脚本实用程序以及管理脚本的默认安装位置）的概念信息。

  • IIS Admin 对象
  • Cscript.exe

Internet 信息服务 Admin 对象

管理脚本自动操作一些最普通的管理任务并演示 IIS Admin 对象的功能。可以用它们来创建和控制 Web 站点、应用程序、目录等等。

可以在使用默认安装到 Inetpub\adminscripts 目录中的 VBScript 文件的过程中运行“管理脚本”，还可以使用 adsutil.vbs 的一个可执行版本（即 adsutil.exe，该文件也是默认安装到目录 Inetpub\adminscripts 中的），以便接受与 adsutil.vbs 同样的参数。 Adsutil.exe 是一个用 C/C++ 编写的示例，它演示如何使用 Microsoft 活动目录服务接口 (ADSI) 处理 metabase。Adsutil.exe 可以从文件中读取命令，而 adsutil.vbs 则没有这个功能。

Cscript.exe

VBScript 管理脚本必须与 Cscript.exe 命令行脚本实用程序一起使用，如果 Cscript 已注册运行 .vbs 文件，则效果更佳。要注册 Cscript，运行 adsutil.vbs 示例。它将为您注册 Cscript.exe。

也可以通过在命令提示中运行下列命令，将 Cscript.exe 注册为默认脚本主机：

cscript //H:cscript
