指定 Web 站点操作员

Web 站点管理员是对 Web 站点具有有限管理权限的 Windows 用户帐号。

添加操作员
  1. 在 Internet 信息服务管理单元中 选择 Web 站点并打开它的属性表。
  2. 在“操作员”属性表的“Web 站点操作员”下，单击“添加”按钮。将打开“添加用户和组”窗口。
  3. 从“名称”列表中选择用户或组，或者从“列出的名称来自”框中选择其他名称列表。
  4. 要从用户组中选择成员，可单击“成员”按钮并从窗口中选择成员。
  5. 要搜寻网络中的用户或组，可单击“搜索”按钮。
删除操作员
  1. 在 Internet 信息服务管理单元中 选择 Web 站点并单击“属性”按钮以显示它的属性表。
  2. 在“操作员”属性表的“Web 站点操作员”下，选择用户或组，并单击“删除”按钮。

注意 Web 站点操作员帐号可以不是 Windows 管理员组的成员。
