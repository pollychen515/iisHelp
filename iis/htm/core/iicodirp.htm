重定向程序请求

作为重定向方法的高级应用，可以将特定目录中文件的所有请求重定向到一个程序。通常，您还可以从原始 URL 中传递任何参数到该程序，使用重定向变量可做到这一点。

将请求重定向到程序
  1. 在 Internet 信息服务管理单元中，选中 Web 站点或目录，然后打开其属性表。
  2. 单击“主目录”、“虚拟目录”或“目录”选项卡。
  3. 选中“重定向到 URL”。
  4. 在“重定向到”框中，键入程序的 URL，包含将参数传递给程序所需的任何重定向变量。

    5. 例如，要将脚本目录中所有的脚本请求重定向到日志程序中（该程序记录请求的 URL 以及与该 URL 一起传递的任意参数），请键入“/Scripts/Logger.exe?URL=$V+PARAMS=$P”。$V 和 $P 是重定向变量。

  5. 选中“上面输入的准确 URL”以防止 Web 服务器将原始文件名添加到目标 URL 中。

有关重定向变量的详细信息，请参阅重定向参考。
