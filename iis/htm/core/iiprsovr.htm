设置 FTP 消息和目录输出样式

当建立 FTP 站点时，可以发送信息消息以便向用户提供关于此站点的详细资料。采用欢迎消息（当用户第一次登录时欢迎用户访问 FTP 站点）和退出消息（当用户注销时显示）的形式。也可以设计消息来提示用户已经达到的最大连接数（FTP 连接限制可以在“FTP 站点”属性表中设置)。

要设置 FTP 欢迎、退出或最大连接数消息
  1. 在 Internet 信息服务管理单元中，选择要设置消息的 FTP 站点。
  2. 右键单击站点并选择“属性”。
  3. 选择“消息”属性表。
  4. 在相应文本框中分别键入欢迎、退出或最大连接数消息。

也可以设置 FTP 站点的目录输出格式。目录输出格式定义了当 FTP 客户发送 LIST 操作时使用的格式；目录输出格式可以是 MS-DOS 或 UNIX。默认目录输出格式是 MS-DOS。

默认情况下，MS-DOS 目录列表样式以两位格式显示年份。您可以更改此设置，以便以四位格式显示年份，方法是启用 FtpDirBrowseShowLongDate metabase 属性。

当文件日期与 FTP 服务器的年份不一致时，UNIX 目录列表样式以四位格式显示年份。如果文件日期与 FTP 服务器年份相同，将不返回年份。

要设置 FTP 站点的目录输出格式
  1. 在 Internet 信息服务管理单元中，选择要设置目录输出格式的 FTP 站点。
  2. 右键单击站点并选择“属性”。
  3. 选择“主目录”属性表。
  4. 在“目录列表格式”下，选择 UNIX 或 MS-DOS。
