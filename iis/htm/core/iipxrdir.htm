MIME 映射

要配置“多用途 Internet 映射扩展 (MIME)”映射，单击“新类型”按钮。在“扩展名”框中输入与文件相关的扩展名。在“内容类型 (MIME)”框中输入 MIME 类型和文件扩展名，格式为：<mime 类型> / <文件扩展名>。

要删除 MIME 映射，选择“已注册的文件类型”下的文件类型，然后单击 “删除”按钮。

如果在主属性表中为您的计算机设置 MIME 映射，那么计算机上的 Web 站点和目录将使用同一映射。可以为一个 Web 站点或目录修改 MIME 映射。只显示已经更改的值，而不显示所有 MIME 映射值。如果再次应用主属性，则 Web 站点或目录的主属性完全代替已更改的属性。也就是说，属性并没有合并。
