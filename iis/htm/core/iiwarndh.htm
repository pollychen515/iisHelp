配置 ASP 应用程序

可以为安装在 Web 服务器上的每个 ASP 应用程序设置某些属性。例如，可以在应用程序中打开会话状态的使用或设置默认脚本语言。应用程序属性将应用于应用程序中的所有 ASP 页，除非在某个单独页中直接覆盖该属性。

配置 ASP 应用程序
  1. 在 Internet 信息服务管理单元中，选择 Web 站点或应用程序的起始点目录。
  2. 打开目录的属性表，然后单击“主目录”、“虚拟目录”或“目录”选项卡。
  3. 单击“配置”按钮，然后单击“应用程序选项”选项卡。
  4. 为应用程序设置属性。有关各种属性的描述，请参阅应用程序选项属性页。
