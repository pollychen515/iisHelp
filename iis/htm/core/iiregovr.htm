关于注册表

可以使用 IIS 管理单元配置许多 Internet 服务。注册表中未使用 IIS 管理单元配置的配置参数或是为某一特定服务的，或是为 IIS 和所有服务的。

只要有可能，应该使用 IIS 管理单元对 Internet 服务器设置进行更改。要使注册表更改生效，必须重新启动该更改所影响的服务。对于全局项目，应该重新启动所有服务。

警告 不正确地使用注册表编辑器将产生问题，包括 Web 站点或 FTP 站点故障。如果出错，Web 站点或 FTP 站点的配置将损坏。应该仅为不能在用户界面中调整的设置编辑注册表项目，并且在直接编辑注册表时小心使用。

IIS 的早期版本中存在的许多注册表键已经移植到一个称为 metabase 的快速访问的、层次结构的数据库中。其余的注册表键用来开始 Internet 服务并保证与 IIS 的早期版本的向后兼容性。任何写入的新信息都必须写入 metabase 中。详细信息，请参阅 Metabase。

注册表属性表的主键

注册表的属性以表的形式出现。配置信息组织如下：

标识符： 代表注册表属性的名称 (DWORD)。
数据类型： 指出数据的变量类型的 DWORD。
范围： 属性可以取值的范围。
默认： 用户未更改时属性所保持的值。
关于属性和对它所做的修改的含义的说明。
