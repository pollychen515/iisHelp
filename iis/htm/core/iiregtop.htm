注册表

配置注册表保存的值定义了 Windows 操作系统和安装在运行 Windows 的计算机上的服务的工作环境。可以使用图形工具，例如控制面板、Windows 安装或 IIS 管理单元来改变这些值，也可以使用包含在 Windows 中的注册表编辑器 (Regedt32.exe) 实用程序来直接修改配置注册表。

本节包括：

  • 关于注册表：与注册表相关的概念上的主题。
  • 全局注册表项目：用于 Internet 服务全局控制的参数。
  • 服务特有的注册表项目：通过 Internet 服务保存在注册表中的参数，用于特定的服务行为。
  • WWW 服务注册表项目：仅用于 WWW 服务的参数。
  • FTP 服务注册表项目：仅用于 FTP 服务的参数。

本节不包含：

  • metabase；请参阅 Metabase。
  • 例行服务器管理；请参阅服务器管理。
