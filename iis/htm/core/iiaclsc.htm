访问控制

您可以控制允许哪些用户和哪些计算机访问您的 Web 服务器和资源。可以使用 NTFS 和 Web 服务器安全功能分配给特定用户访问目录和文件的权限。

本节包含：

  • 关于访问控制：关于限制访问文件和目录的概述信息。
  • 使用 NTFS 保护文件：关于配置服务器使用 NTFS 安全功能的程序上的信息。
  • 设置目录或文件的 NTFS 权限：关于配置 NTFS 权限的程序上的信息， NTFS 权限定义分配给 Windows 用户帐户的文件或目录的访问级别。
  • 允许或拒绝访问计算机：防止特定的计算机、计算机组或整个网络访问您的 Web 内容。
  • 设置 Web 服务器权限：设置适用于所有用户的 Web 和 FTP 内容限制。
  • 控制数据库访问：保护您的数据库文件被未授权访问的准则。

本节不包含：

  • 配置内容分级特性，提醒用户有关令人反感的 Web 内容，请参阅使用内容分级。
  • 配置您 Web 服务器的安全套接字层 (SSL) 安全功能，请参阅设置服务器上的 SSL。
