跟踪处理器使用

进程帐号是 IIS 5.0 的一项新功能，该功能可以记录 Web 站点使用服务器上 CPU 资源的情况。进程帐号只能在单个服务器上分别启用，同时也只能记录各自服务器上的信息，但这些信息不提供单个应用程序或 CGI 应用程序的使用情况。要启用进程帐号，必须选择 W3C 扩展日至文件格式。进程帐号无法用于 FTP 站点。

启用过程记帐
  1. 选择 Web 站点并打开其属性页
  2. 在“Web 站点”属性页中，单击“属性”。
  3. 在“扩充属性”页中，选择“进程帐号”复选框。
  4. 单击“确定”。
