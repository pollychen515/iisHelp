分级属性表

使用此属性表可以对特定 Web 站点、目录或文件配置 Web 服务器的内容分级特性。

Web 服务器的“Internet 内容选择平台 (PICS)”系统使用由 Recreational Software Advisory Council (RSAC) 开发的分级系统，它根据暴力、裸露、性和粗俗语言的等级来分级内容。

要对 Web 内容应用内容分级，可选择“此资源启用分级”复选框。

要设置内容分级，可在“分类”框中单击分级类别（语言、裸露、性或暴力）。要设置类别中何种级别的内容会遭到禁止，可使用分级标尺。每项设置都显示有所分等级的说明。

要标识对内容进行分级的人，可在“邮件名称”文本框中输入 e-mail 地址。

要设置分级内容的截止日期，可单击“分级截止日期”框右边的按钮。将出现日历控件，可用来指定截止日期。
