使用文本编辑器创建 Web 页

您可以使用记事本等标准文本编辑器，通过键入 HTML 标记来创建 Web 页。HTML 标记是一些格式化的指令，它告诉 Web 浏览器如何显示文档。标签由一对包含在小于号和大于号之间的文本字组成 (< >)。例如，下面的 HTML 代码将 Hello World 这句话用粗体显示在 Web 浏览器上：<B>Hello World</B>。显示为 Hello World。

也可以嵌套多个标签以应用多个属性。因此，为了只对 World 这个单词应用斜体，而对整句话应用粗体，应该这样做：<B>Hello <I>World</I></B>。结果显示为：Hello World。注意应该避免覆盖标签；下面这一个并不是有效的 HTML 语法，根据 Web 浏览器的类型可能会产生不正常的结果：<B>Hello <I>World</B></I>。

使用文本编辑器创建 Web 页
  1. 在标准文本 (ASCII) 编辑器中创建一个基本框架，或模板，给浏览器提供一些重要的 HTML 标题和格式化信息。键入下列标签：
    2. <HTML>
<HEAD>
<TITLE> </TITLE>
</HEAD>
<BODY>

</BODY>
</HTML>
  2. 将 Web 页的标题放在 <TITLE> 标签之间。标题显示在 Web 浏览器的标题栏上而不是显示在页面中。选择标题时，请使用能简要说明该 Web 页内容的句子。
  3. 将想要在 Web 浏览器上显示的正文和其他内容放在 <BODY> 标签之间。
  4. 将 HTML 文件用 .htm 扩展名保存（确保文本编辑器不在您的文档后自动添加一个 .txt 扩展名），然后在 Web 浏览器中打开该文件以查看所做的更改。
  5. 将其他元素或格式添加到 Web 页或格式化上；有关信息，请参阅以下 HTML 标签表格。重复第 4 步以查看所做的更改。

提示 应查看 HTML 规范（Internet 上的许多 Web 站点都有提供）以充分利用并理解 HTML。另外，使用 Web 浏览器的源文件查看功能可以查看几乎所有 Web 页的 HTML 源文件。您可以使用该功能学习其他 HTML 编写者是如何设计他们的 Web 页的。

 

HTML 标签 说明
<P> </P> 段落的起始和结束。
<Hn> </Hn> 标题文本的起始和结束。值 n 表示不同的字体大小，应该是 1 到 6 的整数。
<U> </U> 下划线文本的起始和结束。
<UL> </UL> 项目符号列表的起始和结束。每一条列表项必须以 <LI> 标记开头。
<OL> </OL> 项目编号列表的起始和结束。每一条列表项必须以 <LI> 标记开头。
<LI> 表示一个项目列表。不需要结束标签。
<B> </B> 粗体的起始和结束。
< IMG SRC=image file> 将图形文件插入 Web 页。
< BGSOUND SRC=sound file> 装载页面后播放背景音乐。注意 该特性只有 Microsoft Internet Explorer 4.0 或更高版本才支持。
<FONT SIZE=n FACE=font type> < /FONT > 文本字体特征（如大小、颜色或字体）的起始和结束。
<A HREF=file > </A> 创建到文件中某个位置或其他文档的超级链接。例如，下面将创建一个链接，用户单击它可以查看另一个 HTML 文档：<a href="mainpage.html"> 返回主页</a>
<HTML> </HTML> 表示文件内容是 HTML 语言。通常，它们是每一个 HTML 文档的第一个和最后一个标签。
<HEAD> </HEAD> 包含 HTML 文档标题信息，例如文档标题和其他描述性的信息。Web 浏览器和 Web 页搜索程序可以使用这些信息。但是不应将文档正文放在标签之内。
<TITLE> </TITLE> 表示 HTML 文档的标题。标题显示在 Web 浏览器的标题栏中，而不在正文中显示。
<BODY> </BODY> 包含要在 Web 页中显示的内容。
