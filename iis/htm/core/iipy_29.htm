操作员属性页

使用此属性表可以定义哪些 Windows 用户帐户对此站点拥有操作员权限。

Web 站点操作员

要添加用户帐户到操作员列表，单击“添加”。

要删除当前选定的用户帐户，单击“删除”。

操作员可以执行有限的 Web 站点管理功能，包括：

  • 设置 Web 服务器访问权限和日志。
  • 更改默认 Web 文档和页脚。
  • 启用 Web 内容截止日期、HTTP 头和内容分级特性。

具有操作员权限的用户不能执行限制的管理功能，如：

  • 更改 Web 站点标识。
  • 重新配置匿名用户名和密码。
  • 限制带宽。
  • 创建虚拟目录或更改虚拟目录路径。
  • 选择或取消选择应用程序孤立。
