服务器端包含

“服务器端包含 (SSI)”命令指示 Web 服务器将另一文件的内容插入到 Web 页中。使用被包含文件是一种便利方法，可以将多个页使用的信息存储到一个单独文件中。例如，可以使用 SSI 命令将版权声明插入到 Web 站点的所有文档中。

本节包含：

  • 关于服务器端包含：解释服务器端包含命令。
  • 启用服务器端包含：解释如何开启 SSI 命令的处理。
  • 使用服务器端包含指令：解释如何将 SSI 命令插入到 HTML 页中。
  • 服务器端包含参考：使用 SSI 命令的参考信息。
