使用内容分级

可以配置 Web 服务器的内容分级功能，将说明性标签嵌入到 Web 页的 HTTP 头中。某些 Web 浏览器（如 Microsoft Internet Explorer 4.0 或更高版本）可以检测到这些内容标签并帮助用户识别潜在的、令人反感的 Web 内容。Web 服务器默认的基于分级系统的 Internet 内容选择平台 (PICS) 使用由娱乐软件咨询理事会 (RSAC) 开发的系统，该系统根据暴力、暴露、色情及侵犯性语言的级别来对内容进行分级。在设置 Web 内容分级之前，您应该填写 RSAC 问卷以获取指定的 Web 内容建议的内容等级。

对内容进行分级
  1. 在 Internet 信息服务管理单元中，选择 Web 站点、目录或文件，并打开其属性表。
  2. 单击“HTTP 头”选项卡，然后单击“内容分级”下的“编辑分级”。
  3. 在“分级服务”属性页中，在“RSAC 注册”下，单击“分级问卷”。根据 RSAC 提供的指示进行操作。
  4. 单击“确定”。

 

设置内容分级
  1. 在 Internet 信息服务管理单元中，选择 Web 站点、目录或文件，并打开其属性表。
  2. 单击“HTTP 头”选项卡，然后单击“内容分级”下的“编辑分级”。
  3. 在“分级”属性页中，选中“此资源使用分级”复选框。
  4. 在“类别”框中，单击分级类别。使用分级标尺为类别中潜在的、令人反感的资料设置级别。每项设置都显示有所分等级的说明。
  5. 在“对此内容分级人员的电子邮件名”中，键入对内容进行分级的人的电子邮件名，然后选择“到期于”列表并从日历中选择一个分级期满日期。
  6. 单击“确定”。
