#flastmod

#flastmod 命令指示 Web 服务器将指定文件的修改时间插入到 HTML 页中。必须使用 HTML 注释定界符将指令括起来。

该指令仅可用于 HTML 页而无法用于 ASP 页。

语法

<!-- #flastmod PathType = FileName -->

参数

PathType

将 FileName 的路径类型。路径可为以下某种类型：

路径类型 含义
文件 该文件名是带有 #flastmod 命令的文档所在目录的相对路径。
虚拟 文件名为所在 Web 站点虚拟目录中的完全路径。

FileName

指定所需的最后修改时间的文件名。FileName 必须包含文件名扩展，而且必须将文件名用引号 (") 引起来。

注释

包含 #flastmod 命令的文件必须使用映射到SSI 解释器的文件扩展名；否则，Web 服务器将不处理该命令。默认情况下，扩展名 .stm、.shtm 和 .shtml 将映射到解释器 (Ssinc.dll)。如果安装了 Internet 服务管理器，则可以修改默认扩展映射并添加新的映射。请参阅设置应用程序映射。

默认情况下，按“星期名称”、“月份名称”、“日期号”和“年份”格式表示修改时间（例如，Tuesday June 3 1997）。可以使用带有 timefmt 选项的 #config 命令更改格式。

示例

<!-- #flastmod file = "mydocument.htm" -->
