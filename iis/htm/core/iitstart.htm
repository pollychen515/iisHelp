使用 IIS 快速建立站点

当安装 Windows 2000 Server 时，IIS 将创建默认的 Web 和 FTP 站点。本主题描述了如何在那些默认站点上发布信息。

在 Web 站点上发布内容
  1. 为 Web 站点创建主页。关于 Web 站点创建过程中可使用的工具的详细信息，请参阅选择创作工具。
  2. 将主页文件命名为 Default.htm 或 Default.asp。
  3. 将主页复制到 IIS 的默认 Web 发布目录中。默认 Web 发布目录也称为主目录，安装程序提供的位置是 \Inetpub\Wwwroot。
  4. 如果网络具有名称解析系统（通常为 DNS)，那么访问者可以简单地在其浏览器地址栏中键入计算机名到达站点。如果网络没有名称解析系统，那么访问者必须键入计算机的数字 IP 地址。详细信息，请参阅关于名称解析。
在 FTP 站点上发布内容
  1. 将文件复制或移动到默认的 FTP 发布目录。安装程序提供的默认目录是 \Inetpub\Ftproot。
  2. 如果网络具有名称解析系统（通常为DNS），那么访问者可以在其浏览器地址栏中键入后面跟有计算机名的 ftp:// 到达站点。如果没有，那么访问者必须键入 ftp:// 和计算机的数字 IP 地址。要自定义 FTP 站点的外观，请参阅设置 FTP 消息和目录输出样式。
将额外的 Web 或 FTP 站点添加到计算机
  1. 查阅关于名称解析主题中通过分配端口、地址和主机头名维护多个站点一节描述的在一台计算机上维护多个站点的概念信息。
  2. 关于如何添加站点的过程信息，请参阅添加站点。
