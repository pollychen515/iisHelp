性能属性页

使用该属性页可设置影响内存和带宽使用的属性。

调整

将该设置调节为所希望站点每日进行的连接数。如果该数字稍高于实际连接数量，则可加快连接速度并提高服务器性能。如果该数字比实际连接数量高得多，将造成服务器内存的浪费，从而降低服务器总体性能。

启用进程限制

选择该选项可限制 CPU 用于 Web 站点处理应用程序之外的处理时间百分比。如果只选中该框而未选中“强制性限制”，则当分配的限制超出时仅将事件写入“事件日志”。

如果选择了“强制性限制”，可导致运行超出限制。如果站点超出所分配使用范围的 150%，应用程序进程将设置为空闲。如果站点超出所分配使用范围的 200%，Web 站点上的所有进程外应用程序都将停止。
