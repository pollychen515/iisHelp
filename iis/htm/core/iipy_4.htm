IP 地址访问限制

使用此对话框可以允许或禁止特定用户访问此 Web 站点、虚拟目录或目录。

默认情况下，所有计算机都：

授权访问

选中此按钮将允许所有计算机访问此资源，但专门列出的计算机除外。例如，可以通过禁止从特定 IP 地址访问服务器来排除特定的个人或组，或者可以禁止整个网络访问服务器，但允许其它所有人访问。

拒绝访问

选中此按钮将禁止所有远程用户访问，但那些已获特殊授权的 IP 地址用户除外。IP 地址安全在 Internet 上非常有用，可排斥已知用户之外的任何人。

以下所列除外：

要授权或拒绝访问特定的计算机或计算机组：

  1. 单击“添加”。如果选中“授权访问”按钮，则出现“拒绝以下访问”对话框；如果选中“拒绝访问”按钮，则出现“授权以下访问”对话框。
  2. 在“IP 地址”列中，键入被授权或拒绝访问站点的特定计算机的唯一数字 TCP/IP 网络地址。
  3. 在“子网掩码”列中键入数值，该数值可用来标识被授权或拒绝访问站点的计算机组的 TCP/IP 网络细分。
  4. 在“域”列中，键入被授权或拒绝访问站点的网络域名。

注意 虽然编辑 IP 地址限制时，“IP 地址”、“子网掩码”和“域”三个文本框都可用，但不能同时使用。

要限制单个计算机的访问，指定 IP 地址即可。

要限制使用 IP 范围的计算机组的访问，指定 IP 地址和子网掩码即可。

要限制使用域的计算机组的访问，指定域即可。
