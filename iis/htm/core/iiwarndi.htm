终止独立的应用程序

在无须终止 Web 服务器的情况下就可以终止和从内存中卸载应用程序。这使得开发和调试应用程序变得更为容易。

停止独立应用程序
  1. 在 Internet 信息服务管理单元中，选择 Web 站点或应用程序的起始点目录。
  2. 打开目录的属性表，然后单击“主目录”、“虚拟目录”或“目录”选项卡。

    3. 如果在列为“开始点”目录的目录中，则需填写“应用程序名称”对话框。

  3. 单击“卸载”按钮。

    4. 如果“上载”按钮无效，则表明没有位于应用程序的起始点目录。
