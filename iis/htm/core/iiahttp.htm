远程管理

有两个选项可用来远程管理 IIS。要通过 Internet 或代理服务器远程管理 IIS，应使用基于浏览器的 Internet 服务管理器 (HTML)。要通过 intranet 远程管理 IIS，可以使用基于浏览器的 Internet 服务管理器 (HTML) 或 Microsoft 管理控制台 (MMC) 上的 IIS 管理单元。

注意 可以使用 Microsoft 终端服务通过网络连接（如 LAN、PPTP 或拨号）来远程管理 IIS。

要启用基于浏览器的 Internet 服务管理器 (HTML)
  1. 在本地计算机的 IIS 管理单元中，打开“管理 Web 站点”的属性页。
  2. 请留意“Web 站点”属性页上的 TCP 端口号。
  3. 在“目录安全”属性页中，单击“IP 地址和域名限制”下的“编辑”按钮，以便设置将用来远程管理 IIS 的计算机的权限。

 

启动基于浏览器的 Internet 服务管理器 (HTML)
  • 启动浏览器并键入管理站点的域名和指定的端口号。例如，http://www.microsoft.com:端口号/。

    • 注意 在启动 IIS 管理单元之前，Internet 服务管理器 (HTML) 中不会安装日志模块。

作为 Web 站点操作员远程访问 Internet 服务管理器 (HTML)
  • 启动浏览器并键入 Web 站点的域名，后面接“/iisadmin/”。例如，http://www.microsoft.com/iisadmin/。

 

要使用 IIS 管理单元管理 Intranet 上的远程站点
  1. 在网络中任何运行 Windows 的计算机上启动 IIS 管理单元。
  2. 要查找所管理网络中安装的 IIS，请单击工具栏上的“计算机”图标。

    3. 注意 如果没有安装 TCP/IP 和 Windows Internet 命名服务 (WINS)，则只能看到与当前使用计算机所在的相同子网中安装的 IIS。

 

访问本地或远程联机文档
  • 启动浏览器并连接到 http://servername/iishelp/iis/misc/default.asp，其中 servername 是运行 IIS 的计算机。
