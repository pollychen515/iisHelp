文档属性表

使用此属性表可以定义站点的默认 Web 页并在站点文档中附加页脚。

启用默认文档

如果选中此复选框，当浏览器请求没有指定文档名称时，将提供此默认文档。默认文档可以是目录主页或包含站点文档目录列表的索引页。

添加新的默认文档，在文本框中键入文档名称。可以指定多个默认文档。按照在列表中显示的名称顺序依次提供默认文档。服务器返回找到的第一个文档。如果指定多个默认文档，文档名称之间用逗号分隔。

启用文档页脚

选择此选项可以在 Web 服务器发送的每个文档中自动附加 HTML 格式的页脚。

页脚文件不应该是完整的 HTML 文档。它只应当包含必要的 HTML 标记以便格式化页脚内容的外观和功能。

要指定页脚文件的完整路径和文件名，可单击“浏览”。
