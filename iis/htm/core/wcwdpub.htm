发布和管理文件

本节讲述如何连接到 WebDAV 发布目录，通过将文件从计算机拖放到发布目录发布文档及在目录中复制文件。

本节包括：

  • 通过 Windows 2000 发布：介绍如何使用“网上邻居”中 Web 文件夹功能连接到 WebDAV 服务器。
  • 通过 Internet Explorer 5 发布：介绍如何在 Internet Explorer 5 中连接到 WebDAV 服务器及复制文件。
  • 通过 Office 2000 发布：介绍如何在 WebDAV 虚拟目录中直接管理和编辑文件。
    •

注意 即使用户从防火墙后面连接，在具有相应权限及防火墙配置为允许发布的情况下，用户也可以在 WebDAV 目录中发布。
