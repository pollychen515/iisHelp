自定义 W3C 扩充日志记录

如果在 Web 或 FTP 站点上使用 W3C 扩展日志文件格式，就可以通过选择所要记录的字段（项目）进行自定义。可以使用那些对您很重要的字段来收集详细资料，省略不需要的字段以限制日志文件的大小。

自定义 W3C 扩展日志记录
  1. 选中 Web 或 FTP 站点并打开其属性表。
  2. 如果日志记录未启用，请启用它并选择 W3C 扩展日志文件格式。
  3. 单击“属性”。
  4. 在“扩展属性”属性页上，选择要记录的字段。默认情况下，启用“时间”、“客户 IP 地址”、“方法”、“URI 资源”和“HTTP 状态”。
  5. 单击“应用”。
