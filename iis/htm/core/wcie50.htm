通过 Internet Explorer 5 发布

可在 Windows 2000、Windows NT 4.0, Windows 95 或 Windows 98 操作系统上通过 Internet Explorer 5 与 WebDAV 目录连接。建立连接之后，就可以处理文件并将文件发布到该目录，与通过 Windows 2000 建立连接之后一样。具体步骤如下：

  1. 从 Internet Explorer 5 的“文件”菜单单击“打开。
  2. 在“打开”框中键入希望与之建立连接的 WebDAV 目录的 URL。
  3. 选中“以 Web 文件夹方式打开”框，再单击“确定”。

    4. 该目录下随即出现文件列表。
