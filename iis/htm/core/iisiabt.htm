关于服务器端包含

可以使用服务器端包括 (SSI) 命令在把内容发送到浏览器之前将文本、图形或应用程序信息包括到 Web 页中。例如，可以使用 SSI 包含时间/日期标记、版权声明或供客户填写和返回的表单。包含文件是将文本或图形重复并入多个文件的一种简便方法。可以使用简单语句指示 Web 服务器插入文件，而不必在每个文件中键入信息。

因为包含 SSI 命令的文件要求特殊处理，所以必须赋予所有 SSI 文件以 SSI 文件扩展名。默认扩展名是 .stm、.shtm 和 .shtml。可以将另外的扩展名添加到 Web 服务器可识别的列表中；请参阅设置应用程序映射 。

Web 服务器在处理 Web 页的同时处理 SSI 命令。当其处理到 SSI 命令时，将把被包含文件的内容直接插入到 HTML 页中。如果被包含文件也包含 SSI 命令，该命令包含的文件也将被插入。

除用于包含文件的基本命令外，还可以使用 SSI 命令插入有关文件的信息（如文件的大小）或运行应用程序和外壳命令。

详细信息，请参阅服务器端包含参考。
