性能调整

本节介绍如何选择和使用 Windows 2000 工具来监视服务器和站点性能；以及如何调节许多设置来提供整体性能。IIS 提供两种方法访问性能设置。第一种方法是使用 Internet 信息服务管理单元。第二种方法是通过编程直接更改 metabase 中的设置，metabase 是存储 IIS 设置的一种数据结构。本节只介绍通过 Internet 信息服务管理单元更改的设置。

本节包括：

  • 关于性能调整：描述性能问题的概念信息。
  • 内存管理：平衡内存要求与磁盘访问速度。
  • 处理器利用率：找到处理器瓶颈，并限制处理器使用的应用程序。
  • 网络容量：设计网络容量，并最大化连接性能。

本节不包括：

  • 程序管理；请参阅用程序管理 IIS。
  • 关于站点内容选择的性能效果，请参阅配置应用程序。
  • 跟踪 Web 站点通讯；请参阅记录站点活动。
