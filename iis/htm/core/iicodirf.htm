为 Web 页添加页脚

可以配置 Web 服务器自动将一个 HTML 格式配置的文件插入到所有由您的 Web 服务器发送出去的 Web 页底部。例如，为在 Web 页中添加纯文本消息和徽标图案，该文件可以包含 HTML 格式的说明。详细信息，请参阅关于 Web 站点管理和服务器端包含。

给所有的 Web 页添加页脚
  1. 创建 HTML 页脚文件并将其保存到 Web 服务器的硬盘上。

    2. 注意 页脚文件不应该是完整的 HTML 文档。它应该只包含用于格式化页脚内容外观所需的那些 HTML 标签。例如，对用于将公司名添加到所有 Web 页底部的页脚文件，应该包含文本和定义文本字体，及颜色格式所必需的 HTML 标签。

  2. 在 Internet 信息服务管理单元中， 选择 Web 站点、目录或文件，并打开其属性表。
  3. 单击“文档”选项卡，然后选中“启用文档页脚”复选框。
  4. 在文本框中，键入页脚文件的完整本地路径，也可以单击“浏览”按钮查找该页脚文件。

注意 文档页脚可能会降低 Web 服务器的性能，尤其是在 Web 页访问频繁时。
