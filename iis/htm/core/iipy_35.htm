进程选项属性表

使用此属性表可以更改在单个进程中运行的应用程序的设置。可以使用此属性表来更改两种类型进程的设置：Web 服务器进程和单独的应用程序进程。Web 站点选项设置影响在 Web 服务器进程中运行的所有应用程序。单独应用程序的选项设置只影响该应用程序本身。

不成功的客户请求写入事件日志

选择此复选框可以指示 Web 服务器将不成功的客户请求写入 Windows 事件日志。

启用调试异常处理

如果选中此复选框，当 ASP 无法创建 ActiveX 组件（“Server.CreateObject”调用）提供的对象例程时，ASP 将发送一般错误消息。当此复选框没有选中时，ASP 传送来自于组件的特定错误消息。调试组件时撤消此选项。完成调试后，重新选中此选项。

高速缓存的脚本引擎数

指定 Active Server Pages 在内存中高速缓存的 ActiveX 脚本引擎的最大数。

脚本文件高速缓存选项

使用此选项可以指定高速缓存在内存中的预处理的 .asp 文件数。增加高速缓存的文件数可以改善 Active Server Pages 的性能。要高速缓存所有 .asp 文件，可选择“所有请求的 ASP 文件放入高速缓存”。要指定高速缓存脚本文件的最大数，可选择“放入高速缓存的最多 ASP 文件数”，并在文本框中键入值。

CGI 脚本超时

可以设置或更改超时值，它是 CGI 脚本在操作停止前执行并返回值的时间。

注意：可以在 Web 站点、虚拟目录和目录级别上设置应用程序配置属性。
