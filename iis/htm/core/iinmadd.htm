获得 Internet 名称和地址

要使用户能够从 Internet 访问您的站点，必须完成下列步骤：

  1. 建立到 Internet 的连接 从 Internet 服务提供商 (ISP) 或电话公司租用线路。提供商为您的站点安装一条电缆，并接入您计算机的网络适配卡中。通常还要在计算机和 ISP 之间安装一个路由器。在寻找提供商时，确保要了解清楚提供的连接类型以及供应商是否保证最小带宽。同时，也要检查提供的安全类型。
  2. 获得 IP 地址 ISP 为您提供 IP 地址、Internet 网关（默认网关）以及子网掩码。
  3. 注册域名 ISP 可以为您注册域名，或者您也可自己注册。InterNIC 在最初注册域名时收取一定的费用，两年以后将每年收取维护费以便继续在他们的域名服务器中列出您的域名。如果您选择让 ISP 注册域名，那么除了支付 InterNIC 的费用以外，可能还得向 ISP 支付一笔额外的费用。
