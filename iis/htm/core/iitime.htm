超时参考

Internet 信息服务有几个超时值，它们是在安装 Windows 2000 时默认设置的。超时值允许服务器指定将服务器资源分配给特定任务或客户的时间。下表列出了 IIS 中的不同超时值，以及在何处可以对其进行设置。

ADSI Metabase 默认值（秒） 在 IIS 管理单元中有效
AspScriptTimeout MD_ASP_SCRIPTTIMEOUT 90 是
AspSessionTimeout MD_ASP_SESSIONTIMEOUT 1200（20 分钟) 是
ConnectionTimeout MD_CONNECTION_TIMEOUT 900（15 分钟) 是
CGITimeout MD_SCRIPT_TIMEOUT 300（5 分钟) 是
ServerListenTimeout MD_SERVER_LISTEN_TIMEOUT 120（2 分钟) 否
PoolIdcTimeout MD_POOL_IDC_TIMEOUT 30 否
AspQueueTimeout MD_ASP_QUEUETIMEOUT -1（无超时） 否
CpuLimitLogEvent MD_CPU_LIMIT_LOGEVENT 用户定义的百分比 是
CpuLimitPriority MD_CPU_LIMIT_PRIORITY CpuLimitLogEvent 值的 150% 是
CpuLimitProcStop MD_CPU_LIMIT_PROCSTOP CpuLimitLogEvent 值的 200% 是
CpuLimitPause MD_CPU_LIMIT_PAUSE 0（无限） 否

IIS 重新启动超时 默认值（秒）
停止 60
重新启动 20
重新启动 0
