#config

#config 命令指定用于返回客户浏览器的错误消息、日期和文件大小的格式。必须使用 HTML 注释定界符将指令括起来。

该指令仅可用于 HTML 页而无法用于 ASP 页。

语法

<!-- #config Output = String -->

参数

输出

指定要进行格式化的输出。输出可以是下列形式之一：

输出 含义
ERRMSG 当 SSI 命令处理过程中出现错误时控制返回客户浏览器的消息。默认情况下，错误消息将提供详细描述出错情况的调试信息。如果取消这些详细情况，将提供短小、简单的错误消息，例如“出现 SSI 错误”。String 参数包含新的错误消息。
TIMEFMT 指定返回客户浏览器的日期和时间应使用的格式。String 参数指定格式。可通过使用 String 中格式化符号提取日期的单独部分，例如星期几或者月份。（格式化符号与用于 ANSI C strftime 函数的相同）。

%a

星期中日子的缩写（例如，Mon）。

%A

星期中日子的完整名称（例如，Monday）。

%b

月份的缩写（例如，Feb）。

%B

完整的月份名称（例如，February）。

%c

与区域设置对应的日期和时间表示方法（例如，05/06/01 12:51:32）。

%d

十进制数表示的月中的日子 (01-1)。

%H

24 小时格式表示的小时 (00-23)。

%I

12 小时格式表示的小时 (01-2)。

%j

十进制数表示的年中的日子 (001-66)。

%m

十进制数表示的月份 (01-2)。

%M

十进制数表示的分钟 (00-9)。

%p

针对 12 小时格式的当前区域设置的 A.M. 或 P.M. 指示器（例如，PM）。

%S

十进制数表示的秒 (00-9)。

%U

十进制数表示的年中的星期，以星期天作为每星期的第一天 (00-1)。

%w

十进制数表示的星期中的日子，以星期天作为第一天 (0-)。

%W

十进制数表示的年中的星期，以星期一作为每星期的第一天 (00-1)。

%x

当前区域设置的日期表示方法（例如，05/06/01）。

%X

当前区域设置的时间表示法（例如，12:51:32）。

%y

十进制数表示的两位数的年（例如，01）。

%Y

十进制数表示的四位数的年（例如，2001）。

%z, %Z

时区名或缩写；如果时区未知则没有字符。

%%

百分比符号。
SIZEFMT 以千字节或字节为单位显示文件大小。String 参数可以有两个值：ABBREV 以千字节为单位显示文件大小，而 BYTE 则以字节为单位显示文件大小。

注释

包含 #config 命令的文件必须使用映射到 SSI 解释器的文件扩展名；否则，Web 服务器将不处理该命令。默认情况下，扩展名 .stm、.shtm 和 .shtml 将映射到解释器 (Ssinc.dll)。如果安装了 Internet 服务管理器，则可以修改默认扩展映射并添加新的映射。请参阅设置应用程序映射。

示例

<!-- 设置错误消息。 -->
<!-- #config errmsg="出现 SSI 错误"。 -->

<!--以 HH:MM:SS 格式显示时间（例如，23:59:59）。 -->
<!-- #config timefmt="%H:%M:%S" -->

<!--以 MM/DD/YY 格式显示日期（例如，06/28/01）。 -->
<!-- #config timefmt="%m/%d/%y" -->

<!-- 以字节为单位显示文件大小。 -->
<!-- #config sizefmt="字节" -->
