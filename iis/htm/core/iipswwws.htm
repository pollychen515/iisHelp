启用内容截止日期

如果 Web 站点中有时间敏感信息，可以配置设置来保证废弃的信息不被发布。通过使用“HTTP 头”属性表，可以将 Web 站点内容配置为在任何时间点自动过期。当启用内容截止日期时，Web 浏览器将比较当前日期和截止日期，以便决定是显示高速缓存页还是从服务器请求更新的页。

设置 Web 站点内容的截止日期
  1. 在 Internet 信息服务管理单元中，选择要设置内容截止日期的 Web 站点、虚拟目录、目录或文件。
  2. 右键单击站点、虚拟目录、目录或文件，并选择“属性”。
  3. 选择“HTTP 头”属性表。
  4. 选择“启用内容截止日期”复选框。
  5. 选择“立即过期”，“在此后过期”或“过期”单选按钮并在对应框中输入适当的过期信息。

注意 通常，时间敏感信息仅限制单个文件、目录或 Web 站点；然而，通过配置“WWW 主属性”，也可以为计算机上的所有 Web 站点设置内容截止日期。详细信息，请参阅更改继承的默认设置。
