获取客户证书

可以从称为“证书颁发机构”的、可信的第三方机构获取客户证书。在签发证书之前，颁发机构可能需要验证您的身份。颁发机构要求的标识信息取决于所要获取的证书类型。如何获取并执行这些证书取决于所使用的浏览器。本主题中的过程仅适用于 Internet Explorer 4.0 或更高版本。有关其他浏览器客户证书的颁发过程，请参阅浏览器文档。

获取客户证书
  1. 选择证书颁发机构。选择证书颁发机构时请考虑以下几方面的问题：
    • 该证书颁发机构是否知名可信？
    • 为验证您的身份，颁发结构需要从您处得到何种类型的信息？大多数证书颁发机构需要您提供详细的身份证明信息，例如姓名、地址、公司名和电子邮件地址等。
    • 证书颁发机构是否有 Web 站点可供申请客户证书？
    • 所签发的证书是否与所使用的浏览器和服务器兼容？

    要取得为 Internet 信息服务提供证书签发服务的证书颁发机构列表，请参阅获取服务器证书。

  2. 请参考由证书颁发机构提供的专门订购说明。
  3. 订购您的证书。一些客户证书只对用来产生证书请求的计算机和 Web 浏览器有效：因此请确保联机请求是从所要安装证书的计算机上发出的。
  4. 收到您的客户证书后，请使用 Web 浏览器的安全功能安装该证书。（有些浏览器可能会将客户证书视为“浏览器”或“个人”证书。）
