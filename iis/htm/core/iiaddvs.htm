添加站点

可以通过启动站点向导来向计算机中添加新站点。

添加新站点
  1. 在 Internet 信息服务管理单元中选择计算机或站点，然后单击“动作”按钮。
  2. 单击“新建”，然后单击“Web 站点”或“FTP 站点”启动站点向导。
  3. 根据屏幕指令指定新站点的标识信息。必须提供端口地址和主目录路径。如果通过主机头文件将其他站点添加到单一 IP 地址，必须指定主机头文件名称。

    4. 注意 在站点向导中，“全部未指定”表示指派给计算机但并未指派给特定站点的 IP 地址。默认的 Web 站点使用尚未指派给其他站点的所有 IP 地址。只能设置一个站点使用未指派的 IP 地址。关于如何标识站点的详细信息，请参阅“关于名称解析”中的通过指定端口、地址和主机头文件名维护多个站点。
