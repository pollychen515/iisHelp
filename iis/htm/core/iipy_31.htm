ISAPI 筛选器属性页

使用此属性表可以设置 ISAPI 筛选器选项。ISAPI 筛选器是在处理 HTTP 请求过程中响应事件的程序。

要添加 ISAPI 筛选器，单击“添加”按钮。在“筛选器”列中输入可识别的筛选器名称，并在“可执行”列中输入筛选器的路径。

要删除 ISAPI 筛选器，选中筛选器并单击“删除”按钮。

要更改 ISAPI 筛选器的属性，选中筛选器并根据需要更改“筛选器”和“可执行”文本框。

要更改 ISAPI 筛选器执行的顺序，选中筛选器并单击“上移”或“下移”按钮。只能更改具有相同优先级的筛选器的加载顺序。

注意 在主属性级配置的 ISAPI 筛选器是全局的，即，它们适用于在服务器创建的所有 Web 站点。在主属性级配置的 ISAPI 筛选器可能与在单独 Web 站点级配置的 ISAPI 筛选器不同。另外，在 Web 站点级不会显示全局筛选器。
