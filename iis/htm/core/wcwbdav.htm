WebDAV 发布

本节解释了如何在 IIS 5.0 服务器上设置一个 WebDAV 发布目录，并告之客户端如何连接到服务器以编辑和处理文件。

本节包括：

  • 关于WebDAV：解释了 WebDAV 如何创建目录供远程用户发布和处理文件。
  • 创建发布目录：如何设置 WebDAV 发布点目录。
  • 管理 WebDAV 安全：提供一些优化 IIS 5.0 和 Windows 2000 安全以保护您的 WebDAV 站点的技巧。
  • 搜索 WebDAV 目录：如何配置 WebDAV 目录以便用户可以从中搜索内容和文档属性。
  • 发布和管理文件：如何通过 Windows 2000、Internet Explorer 和 Office 2000 发布内容和处理文件。

注意 WebDAV 是 HTTP 1.1 提议的草案的实现，因此不能用于非 HTTP 服务，如 FTP 站点。同时，这一实现当前只用于 IIS 5.0。
