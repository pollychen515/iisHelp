证书

证书是一种数字标识，用于服务器和从服务器请求信息的客户。它们的功能有如护照，或其他官方的身份证（这些证件用于标识其持有人身份）。证书是 IIS 安全套接字层 （SSL） 功能的一部分，用于建立安全连接，通过该连接可以发送敏感信息。

本节包含：

  • 关于证书： 证书的总体概述。
  • 设置服务器上的 SSL： 如何在服务器上建立和管理安全套接字层 (SSL)。
  • 使用新的安全任务向导：证书、CTL 和权限向导中可用功能的概述。
  • 获取服务器证书：学习如何获取需要使用 SLL 的服务器证书
  • 使用证书信任列表：将新的证书颁发机构添加到服务器上的信任签发部门的列表中。
  • 获取客户证书： 如何获取用户数字标识（客户证书）。
  • 启用客户证书： 启用客户证书 SLL 验证的过程。
  • 将客户证书映射到用户帐户：使用客户证书直接将用户连接到 Windows 用户帐户。

本节不包含：

  • 控制用户访问您的 Web 服务器文件和目录的方法；请参阅访问控制。
  • 有关设置或更改加密方法的信息，请参阅加密。
