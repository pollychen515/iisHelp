通过 Windows 2000 发布

从 Windows 2000 Professional、Windows 2000 Server 或 Windows 2000 Advanced Server，可以连接到其他服务器上的 WebDAV 发布目录。要作到这一点，请执行下面的步骤：

  1. 双击桌面上的“我的网络位置”图标。
  2. 双击“添加网络位置”。
  3. 在“添加网络位置向导”中，键入要连接 WebDAV 目录的 URL。
  4. 单击“下一步”。
  5. 键入要连接目录的友好名称。
  6. 单击“完成”。
  7. 双击“我的网络位置”。

    8. 将会看到 WebDAV 站点 Web 文件夹的友好名称。例如，如果将“My Publishing Folder”作为友好名称，将会看到“快捷方式 My Publishing Folder”。

  8. 双击 Web 文件夹名称。

    9. 依据示例中前面的步骤，双击“快捷方式 My Publishing Folder”。将会看到相应目录中文件的列表。

注意 在默认情况下 IIS 5.0 并不安装到 Windows 2000 Professional 中。因此，如果您的服务器运行 Windows 2000 Professional 并且在安装选项中没有选中 IIS 5.0，您需要手工进行安装。要达到此目的，请执行下列步骤：

  1. 单击“开始”，指向“设置”，然后单击“控制面板”并启动“添加/删除程序”。
  2. 在“添加/删除程序”对话框左面的列中，单击“添加/删除 Windows 组件”。
  3. 出现“Windows 组件向导”后，单击“下一步”。
  4. 在“Windows 组件”列表中选中“IIS”。
  5. 单击“下一步”，然后根据指导操作。
