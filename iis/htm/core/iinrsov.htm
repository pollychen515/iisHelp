名称解析

要使访问者能到达站点并查看网页，站点在 Internet 或本地 Intranet 上必须具有唯一的标识。如何将站点“友好名称”解析为其它计算机可访问的地址取决于网络上可用的名称解析系统。

本节包括：

  • 关于名称解析：可用的各种命名系统、各系统的工作原理、根据情况选择哪个系统、以及分配站点名称与 IP 地址的规则。
  • 获取 Internet 名称和地址：如何获得 Internet 上的注册域名。
  • 验证计算机标识：检查 Windows 计算机名称和 IP 地址。

本节不包括：

  • 关于分配 Web 站点标识信息的说明，请参阅命名 Web 站点。
