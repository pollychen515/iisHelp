服务器端包含参考

欢迎访问服务器端包含命令的参考页。可以使用服务器端包含命令将文件内容包含在 HTML 页中，以及包含有关文件的信息（如文件大小）。也可以在 ASP 中使用某些服务器端包含命令。

本节包含：

  • #config：指定用于返回客户浏览器的错误消息、日期和文件大小的格式。
  • #echo：将环境变量值插入到 HTML 页中。
  • #exec：运行应用程序或外壳命令并将输出结果插入到 HTML 页中。
  • #flastmod：将文件修改时间插入到 HTML 页中。
  • #fsize：将文件大小插入到 HTML 页中。
  • #include：将文件包含在 HTML 或 ASP 页中。
