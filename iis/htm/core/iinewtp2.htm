管理

本节说明如何使用 Internet 信息服务管理单元管理 IIS。IIS 管理单元驻留于 Microsoft 管理控制台 (MMC)，它是一个强有力的站点管理工具，可提供对所有服务器设置的访问。使用 IIS 管理单元可管理企业内部网中的复杂站点或在 Internet上发布信息。

本节包含：

  • Web 站点管理：创建站点内容并创建必要的发布目录结构。
  • 服务器管理：有关调整默认设置以提高性能或改进安全性的高级主题。
  • 管理员参考：关于注册表和 metabase 设置的详细信息。

注意 Internet 信息服务管理单元是 IIS 5.0 的一个管理工具，它和 Windows 2000 中的其他管理功能集成在一起。在以前的版本中该工具称为 Internet 服务管理器。

本节不包含：

  • 关于建立 Web 或 FTP 站点的信息，请参阅使用 IIS 快速建立站点。
  • 关于安装或安装程序的信息，请参阅安装 IIS。
