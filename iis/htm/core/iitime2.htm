IIS 管理单元中超时设置的位置

该页是对超时表主题的补充。

更改 AspScriptTimeout 值
  1. 在 Internet 信息服务管理单元中，选择 Web 站点或应用程序的起始点目录。
  2. 打开目录的属性表，然后单击“主目录”、“虚拟目录”或“目录”选项卡。
  3. 单击“配置”按钮，然后单击“应用程序选项”选项卡。
  4. 在“ASP 脚本超时”中，键入期望值。

更改 AspSessionTimeout 值
  1. 在 Internet 信息服务管理单元中，选择 Web 站点或应用程序的起始点目录。
  2. 打开目录的属性表，然后单击“主目录”、“虚拟目录”或“目录”选项卡。
  3. 单击“配置”按钮，然后单击“应用程序选项”选项卡。
  4. 在“会话超时”文本框中，键入期望值。
更改 ConnectionTimeout 值
  1. 在 Internet 信息服务管理单元中，选择 Web 站点或应用程序的起始点目录。
  2. 打开目录的属性页，然后单击“Web 站点”选项卡。
  3. 在“连接超时”文本框中，键入期望值。
更改 CGITimeout 值
  1. 在 Internet 信息服务管理单元中，选择 Web 站点或应用程序的起始点目录。
  2. 打开目录的属性表，然后单击“主目录”、“虚拟目录”或“目录”选项卡。
  3. 单击“配置”按钮，然后单击“进程选项”选项卡。
  4. 在“CGI 脚本超时”文本框中，键入期望值。
记录 Web 站点的处理器使用情况
  1. 在Internet 信息服务管理单元中，选择希望记录其处理器使用情况的 Web 站点。
  2. 打开 Web 站点的属性页，然后单击“性能”选项卡。
  3. 选择“启用进程限制”，然后键入希望该站点被限定的 CPU 时间的百分比。
  4. 如果希望启用 CpuLimitPriority 和 CpuLimitProcStop，请选择“强制性限制”。
