更改主目录

每一个 Web 站点必须有一个主目录。如果在同一台计算机上设置 Web 站点和 FTP 站点，就必须为每一种服务（WWW 和 FTP）指定不同的主目录。WWW 服务的默认主目录是 C:\InetPub\Wwwroot。FTP 服务的默认主目录是 C:\InetPub\Ftproot。可以选择不同的目录作为您的主目录。

更改主目录
  1. 在 Internet 信息服务管理单元中，选中 Web 或 FTP 站点并打开其属性页。
  2. 单击“主目录”选项卡，然后指定主目录的位置。可以选择：
    • 计算机硬盘上的目录。
    • 其他计算机上的共享目录。
    • 重定向到 URL（浏览器请求此 URL 转向另一个新的 URL）。无法重定向 FTP 目录。
  3. 在文本框中，指定目录的路径名、共享名或 URL。

    4. 注意 如果选择了网络共享上的某个目录，有可能需要键入用户名和密码以访问该资源。建议使用 IUSR_computername 帐户。如果使用具有服务器管理员权限的帐户，客户端有权使用服务器操作。这样会严重威胁网络的安全。
