加密

可以使用 Web 服务器 的 SSL 加密功能来编码 Web 服务器传输和接收的信息。当启用 SSL 加密时，可以防止未授权个人解码传输的原始内容。

本节包括：

  • 关于加密：SSL 安全通讯概述。了解加密如何保护通过不安全网络（如 Internet）传输的信息。
  • 启用加密： 启用 Web 服务器 SSL 加密功能的步骤。
  • 设置加密强度：了解如何要求用户建立与 Web 服务器之间的 128 位会话密钥的安全通讯。

本节不包含：

  • 创建 Web 服务器密钥对文件，它用于 SSL 验证；请参阅使用新的安全任务向导。
  • 在使用 SSL 安全功能之前，首先应获得 Web 服务器证书；请参阅获取服务器证书。
