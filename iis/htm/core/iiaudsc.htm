审核

可以使用安全审核技术跟踪用户活动并监测访问 NTFS 目录和文件的非法尝试。可供审核的活动包括：

  • 所有（包括成功和不成功的）登录尝试。
  • 访问限制帐户的用户尝试。
  • 执行限制命令的用户尝试。

本节包括：

  • 关于审核： 学习如何监测和评估 Web 服务器安全。
  • 配置和监视审核：创建安全日志。
  • 侦测未授权的访问：学习如何解释安全日志中的未授权访问的标记。

本节不包含：

  • 如何记录用户的 Web 和 FTP 站点活动，请参阅记录站点活动。
  • 如何控制对 Web 站点和 FTP 站点的访问，请参阅访问控制。
