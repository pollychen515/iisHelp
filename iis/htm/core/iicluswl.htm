复制和群集

要点 本节描述的群集功能仅适用于 Windows 2000 Advanced Server。

复制和群集是一种技术，它使得多服务器的站点可以满足对其不断增长的要求。在本节中，“复制”定义为将内容和配置设置从一个服务器节点复制到另一个服务器节点；群集定义为将两台计算机连接起来以达到共享资源和请求负载的目的。当使用群集软件将计算机组合在一起并像单个计算机一样工作时，可以执行容错、负载平衡和复制。

本部分包括：

  • 关于群集：介绍与群集相关的术语和概念。
  • IIS 中的复制和群集：如何在服务器群集中安装 IIS 并使用 Iissync.exe 实用程序将配置设置从一台计算机复制到另一台计算机。
  • 升级 IIS 群集资源：如何通过标准升级或滚动升级来升级 IIS 群集资源。

本节不包含：

  • 添加 Web 站点到服务器；请参阅添加站点。
  • 优化 Web 或 FTP 服务器性能；请参阅性能调整。
