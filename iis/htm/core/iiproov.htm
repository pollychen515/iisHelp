处理器的利用

随着 Web 服务器的逐渐成熟，其性能已变得日益重要。今天，由于数据库发布、内容检索以及协作计算等 Web 应用的不断发展，如何最大限度地利用硬件和软件性能已经成为了首要的课题。本节将集中讨论处理器各个方面的性能，并根据性能监视测试所得到的反馈对其升级提出一些建议。

本节包含：

  • 关于处理器的使用：性能问题的概念性信息。
  • 限制进程：对 Web 站点所用 CPU 时间总量实施限制。

本节不包含：

  • 有关创建事务和隔离程序的性能方面的问题。请参阅配置应用程序。
  • 有关跟踪 Web 站点通信量，请参阅记录站点活动。
