安全

Web 服务器的正确安全措施可以降低或消除来自怀有恶意的个人以及意外获准访问限制信息或无意中更改重要文件的善意用户的各种安全威胁。

本节将说明如何配置 Web 服务器和 Windows 以便正确保护 Web 站点和执行其他基本安全功能。

本节包括：

  • 关于安全性： Web 服务器安全功能的概念性概述，以及关于启动和学习使用高度安全的安全套接字层 (SSL) 功能的提示。
  • IIS 安全检查表：保护 Web 服务器的建议列表，包括 Windows 设置、IIS Web 权限和物理安全。
  • 验证：确认试图与限制内容建立连接的用户的标识。
  • 访问控制：控制用户如何访问和操作文件与目录。
  • 加密：使用 SSL 加密功能保护信息的保密性。
  • 证书：使用证书和 SSL 功能建立安全连接。
  • 审核：监视安全活动以防止篡改和非法访问。

本节不包含：

  • 隔离服务器应用程序以便它们在独立于 Web 服务器进程的进程中运行；请参阅独立的应用程序。
  • 验证网络名和计算机地址；请参阅验证计算机标识 。
  • ASP 应用程序安全性；请参阅 IIS 应用程序安全分支 。
