高级多 Web 站点配置

使用此对话框可以配置 Web 站点的附加标识。

单击“添加”或“删除”按钮可以添加或删除列出的标识。要编辑现有的标识，请选择 IP 地址并编辑相应的字段“IP 地址”、“TCP 端口”、“SSL 端口”或“主机头名”。

每个 Web 站点必须有唯一的标识特性组合。因此，当多个 Web 站点可以共享三个标识特性（域或主机头名、IP 地址和端口）中的两个时，必须有一个不同的特性。而且，由于 SSL 证书包含域名，所以使用证书的 Web 站点不能与其它 Web 站点共享 IP 地址。

SSL 端口

决定用于建立安全套接字层 (SSL) 连接的端口。

对于 Web 服务器的默认 Web 站点，此端口设置为 443。（如果创建新 Web 站点，必须显式设置此端口。）可以更改 SSL 端口为任何唯一的 TCP 端口号；不过，用户必须事先知道才能请求该端口号，否则，它们的请求不能连接到服务器。只有已经安装了服务器证书，此选项才可用。
