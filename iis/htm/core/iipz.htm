FTP 站点属性表

使用该属性页为服务器设置标识参数。服务器和域名以及 IP 地址是可选的。

标识

说明

可以为服务器名使用想要的任何名称。这个名称主要是用于您自己的标识和组织目的。

IP 地址

对于出现在该框中的地址，它必须已经被定义在“控制面板”中使用。详细信息，请参阅 Windows 文档。

TCP 端口

确定服务正在运行的端口。

默认值是端口 21。可以将端口更改为任何唯一的 TCP 端口号；客户端必须事先知道才能请求该端口号，否则请求不能连接到服务器。

连接

无限

允许同时连接到服务器的连接数没有限制。

限制到

设置允许到服务器的同时连接的最大数量。

连接超时

设置服务器断开与一个非活动的用户的连接之前的时间长度。这将确保在 HTTP 协议无法关闭某个连接时，关闭所有的连接。

启用日志记录

选择该选项以启用 FTP 站点日志记录，它可以记录用户活动的详细信息，并创建不同格式的日志。启用日志后，请从“活动日志格式”列表中选择格式。格式如下：

  • Microsoft IIS 日志格式 一个固定的 ASCII 格式。
  • W3C 扩充日志文件格式 一个自定义的 ASCII 格式，默认为选中。
  • ODBC 日志 一个登录到数据库的固定格式（仅用于 Windows 2000 Server）。

要配置日志文件创建（例如，每周或按文件大小）选项，或者配置 W3C 扩展日志或 ODBC 日志的属性，请单击“属性”。
