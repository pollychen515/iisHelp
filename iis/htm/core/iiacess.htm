IIS 辅助功能

IIS 中指定了下列快捷键（或快捷键命令）。要查看 Windows 所有快捷键的清单，请在 Windows 帮助索引中查找“开始菜单：键盘命令”。要了解关于在 Microsoft 管理控制台中导航的特定信息，请在 Windows 帮助索引中查找“MMC：易访问性”。

要访问 IIS 管理单元
  1. 请打开“开始”菜单 (CTRL+ESC)，然后使用方向键定位，按 ENTER 键打开“设置”、“控制面板”、“管理工具”和“计算机管理”。
  2. 在 MMC 中，打开“服务器应用程序和服务”。
  3. 打开“Internet 信息服务”。
要访问特定的属性表
  1. 在 IIS 管理单元中，使用方向键选择要管理的站点、虚拟目录或文件。要访问文件，请转到详细资料窗格。使用快捷键 ALT 在 IIS 管理单元的控制台树和详细资料窗格间转换。
  2. 按快捷键 SHIFT+F10 打开“动作”菜单选择项目。

    3. 注意 要打开“查看”菜单，可以使用快捷键 CTRL+A。

  3. 使用向上或向下方向键选择“属性”。按 ENTER 键。

    4. 注意 键入“P”将暂停您的站点。

  4. 按快捷键 TAB 直到属性表标题突出显示。
  5. 使用向左或向右方向键在每一行中选择相应的属性表。
  6. 要转换到属性表的其他行，请使用向上或向下方向键。
  7. 要使用某一属性表，请按 TAB 键。可以使用快捷键 TAB 选择项目或使用快捷键 ALT+<带下划线的字母>。

    8. 注意 要选中或清除某一复选框，请选中它并按空格键。

在 IIS 联机文档中查找定位
  1. 如果顶部没有“内容”选项卡，请按快捷键 TAB 键或 SHIFT+TAB 键，直到“内容”选项卡突出显示。
  2. 按 ENTER 键将选项卡置于最前。

    3. 注意 使用本程序，您可以在“内容”、“搜索”和“索引”选项卡之间切换。

  3. 按住 TAB 键直到目录中的一个项目突出显示。
  4. 使用向上或向下方向键选择项目。
  5. 要打开一本书，使用向右方向键。
  6. 要关闭一本书，使用向左方向键。
  7. 要打开一个文档，请选中它并按 ENTER 键。
