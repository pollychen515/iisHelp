#fsize

#fsize 命令指示 Web 服务器将指定文件的大小插入到 HTML 页中。必须使用 HTML 注释定界符将指令括起来。

该指令仅可用于 HTML 页而无法用于 ASP 页。

语法

<!-- #fsize PathType = FileName -->

参数

PathType

将 FileName 的路径类型。路径可为以下某种类型：

路径类型 含义
文件 该文件名是带有 #fsize 命令的文档所在目录的相对路径。
虚拟 文件名为所在 Web 站点虚拟目录中的完全路径。

FileName

指定所需的文件大小的文件名。FileName 必须包含文件名扩展，而且必须将文件名用引号 (") 引起来。

注释

包含 #fsize 命令的文件必须使用映射到 SSI 解释器的文件扩展名；否则，Web 服务器将不处理该命令。默认情况下，扩展名 .stm、.shtm 和 .shtml 将映射到解释器 (Ssinc.dll)。如果安装了 Internet 服务管理器，则可以修改默认扩展映射并添加新的映射。请参阅设置应用程序映射。

默认情况下，以千字节为单位表示大小。可通过使用带有 sizefmt 选项的 #config 命令更改度量单位。

返回的文件大小字符串包含逗号 (,) 作为千位占位符（例如，1,389,587）。

示例

<!-- #fsize file = "myfile.htm" -->
