启用日志记录

可以为每个 Web 和 FTP 站点启用日志记录并选择日志格式。启用日志记录之后，就对所有站点的目录启用了日志记录，但也可以为特定目录禁用日志记录。

启用 Web 或 FTP 站点上的日志记录。
  1. 选中某个 Web 或 FTP 站点并打开其属性页。
  2. 在“Web 站点”或“FTP 站点”属性页中，选择“启用日志”复选框。
  3. 在“活动日志格式”列表中，选择一种格式。默认情况下，“启用日志”复选框已被选中，其格式为“W3C 扩展日志文件格式”，并已启用了以下字段：“时间”、“客户 IP 地址”、“方法”、“URI 源”以及“HTTP 状态”。

    4. 注意 如果选择了 ODBC 日志格式，请单击“属性”，并在相应的框中键入“数据源名”以及数据库表的名称。如果访问数据库需要用户名称和密码，请分别键入并单击“确定”。

  4. 单击“应用”。
  5. 单击“确定”。
禁用或启用站点上特定目录的日志记录
  1. 选择目录并打开其属性页。
  2. 在“主目录”或“目录”属性页中，找到“日志访问”复选框。默认情况下，该复选框已被选中。
  3. 要禁用目录的日志记录，请清除该复选框；要启用日志记录，请选中该复选框。这些设置并不会影响进程帐户。
