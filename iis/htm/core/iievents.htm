IIS 的 Windows 事件

IIS 在“事件查看器”中生成事件，以便用户验证 IIS 服务执行是否正确。“事件查看器”跟踪（按重要程度）信息、警告和错误事件。“事件查看器”中的日志提供所有 Windows 服务和进程的审核记录，如登录时间和连接时间。

本节包括：

  • 关于 Windows 事件：与 Windows 事件相关的主题。
  • Web 服务事件：Web 服务生成的事件列表。
  • FTP 服务事件：FTP 服务生成的事件列表。
  • Active Server Pages 事件：ASP 生成的事件列表。

本节不包含：

  • 错误消息；请参阅自定义错误消息。
  • 日志记录；请参阅记录站点活动。
