自定义错误消息

可以配置 IIS 来发送自定义错误信息，而不是发送默认的 HTTP 1.1 错误消息。这些自定义错误信息可以映射为文件名或 URL。

本节包括：

  • 关于自定义错误消息：关于默认和自定义 HTTP 1.1 错误的总体信息。
  • 添加自定义错误消息：关于向 Web 站点添加自定义错误消息的过程信息。
  • 启用详细自定义错误消息：关于使用 IIS 随带的详细（或友好） HTTP 1.1 错误的过程信息。
  • 启用 ASP 错误处理进程：关于使用 500-100.asp 自定义错误文件来处理 ASP 错误的总体信息。

本节不包含：

  • Web站点内容；请参阅 Web 站点管理。
  • Windows 事件；请参阅 IIS 的 Windows 事件。
