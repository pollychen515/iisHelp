ASP 调试属性表

使用此属性表可以为在所选应用程序中运行的 ASP 脚本设置调试选项。

调试标志

启用 ASP 服务器端调试

选中此复选框，可以允许 Web 服务器在处理 ASP 脚本时进入 Microsoft 脚本调试器。随后即可使用调试器检查脚本。

启用 ASP 服务器端调试将导致 ASP 在单一线程模式运行。由于性能方面的原因，建议您不要在产品环境中启用 ASP 调试。

启用 ASP 客户端调试

此复选框被保留，备以后使用，但并不影响当前版本的 ASP。

脚本错误消息

传送详细 ASP 错误消息给客户

选中此选项可以发送特定调试信息（包括文件名、错误消息和行号）到浏览器。

传送文本错误消息给客户

如果选中此选项，当任何错误防止 Web 服务器处理 ASP 脚本时，将发送默认错误消息到浏览器。特定错误消息被写入错误日志通过在文本框中键入新消息，可以更改默认错误消息。
