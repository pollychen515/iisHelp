命名 Web 站点

每个 Web 站点（虚拟服务器）都有一个描述性名称，并支持一个或多个主机头名。主机头名使得在一台计算机上可维护多个域名。并不是所有浏览器都支持主机头名。Microsoft Internet Explorer 4.0 和 Netscape Navigator 4.0 浏览器以及它们的最新版本都支持主机头名的使用，但这两种浏览器的早期版本不支持。

如果访问者试图使用不支持主机头的旧浏览器连接您的站点，则访问者将定向到分配给该 IP 地址的默认 Web 站点（如启用了默认站点），该站点没有必要是请求的站点。另外，如果浏览器请求的站点目前停止使用，则访问者将收到默认的 Web 页。因此，请仔细考虑默认 Web 站点显示的内容。通常，ISP 将其主页作为默认显示，而不是某个客户的 Web 站点。这可防止对已停止站点的请求转到错误的站点。此外，默认站点可以包含脚本，支持旧的浏览器使用主机头名。

命名 Web 站点
  1. 在 Internet 信息服务管理单元中，选择 Web 站点并打开其属性页。
  2. 在“Web 站点”属性页的“说明”框中，输入该站点的描述性名称。
分配主机头名
  1. 询问系统管理员，将映射到静态 IP 地址的新站点名称添加到名称解析系统（通常是 DNS）。
  2. 在“Web 站点”属性页中，单击“高级”按钮。
  3. 在“高级多 Web 站点配置”属性页中，单击“添加”按钮分配 Web 站点的主机头文件、IP 地址以及端口。

    4. 注意 使用 SSL 时，仅可为每个 IP 地址分配一个主机头名，这是因为域名是在服务器证书中指定的。不过，每个 Web 站点可以拥有多个服务器证书、多个 IP 地址以及多个 SSL 端口。
