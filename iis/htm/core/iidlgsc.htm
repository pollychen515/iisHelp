设置默认登录域

通过基本验证验证的用户必须使用有效的 Windows 用户名和密码登录。用户名通常包括：

  • Windows 域名。
  • 帐户用户名。

这些用来识别 Web 服务器作为单个实体管理的计算机或计算机组。例如，使用 PhilSpencer 帐户登录 Sales1 域的用户将被登录为 Sales1\PhilSpencer。然而，如果用户不提供域名，您可以配置 Web 服务器来为基本验证假定一个默认登录域。

设置默认登录域。
  1. 在 Internet 信息服务管理单元中，选择 Web 站点、目录或文件，然后打开其属性表。
  2. 选择适当的“目录安全性”或“文件安全性”属性表。在“匿名访问和验证控制”下，单击“编辑”。
  3. 在“验证方法”对话框中，选择“基本验证”复选框。

    4. 注意 由于基本验证通过未加密的网络发送密码，因此，会出现一个对话框，询问是否希望继续。单击“确定”以便继续进行。

  4. 单击“编辑”。
  5. 在“基本验证”对话框中，键入或者浏览到新的默认登录域。单击“使用默认值”来使用 Web 服务器的默认域名。
