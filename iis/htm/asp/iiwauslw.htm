建立 ASP 页

ASP 提供强大的可扩展框架，便于使用任何 COM 脚本或编程语言来创建服务器端脚本。本节专门讲述使用脚本语言创建 .asp 文件的基础知识。您将学会如何完成一系列基本的编程任务，从创建循环到管理数据库和处理事务。不管您是编写脚本的新手还是高手，都可以将本节中的主题预想为开发目标，这样，您就可以采用更加复杂的方式利用 ASP，从而倍受鼓舞 。这样最终就能编写出一些性能更加优良，更加易于维护的应用程序。

虽然这些主题介绍了一些脚本和编程概念，但主要目的并不是在于讲述脚本语言。由于 Microsoft 脚本语言（Microsoft VBScript 和 Microsoft JScript）都有自己的文档，各地书店也能买到许多其他脚本的书籍。所以，对于编写脚本的新手，应当充分利用各种书籍、培训以及 Internet 资源，这样，有助于精通这些语言。

本节包括：

  • 创建 ASP 页：描述 ASP 文件并解释如何在网页中添加脚本命令。
  • 使用脚本语言：解释如何设置基本脚本语言以及如何在服务器脚本中使用 VBScript 和 JScript。
  • 使用变量和常量：介绍了如何在 ASP 脚本中使用变量以及如何访问常量定义。
  • 与客户端脚本交互：介绍如何编写各种用于创建客户脚本并与客户脚本交互的服务器端脚本。
  • 编写过程：讲述如何定义过程（函数和子程序）以及通过服务器端脚本调用过程。
  • 使用集合：讲述如何访问内建对象集合中的对象，包括对集合的循环访问。
  • 处理用户输入：解释如何收集和处理从 HTML 表单中收集的信息。
  • 使用组件和对象：解释如何创建 COM 组件对象的实例、如何使用 ASP 内建的对象以及如何使用任何对象的方法和属性。
  • 设置对象作用域：说明对象的作用域，或叫范围，并讲述如何创建 session 和 application 作用域对象。
  • 将内容发送到浏览器：讲述如何控制网页发送到浏览器的方式。
  • 包含文件：解释如何使用“#include”语句将文件内容插入到 .asp 文件。
  • 管理会话：讲述如何保留用户信息。
  • 访问数据源：解释如何使用 ASP 和 Microsoft ActiveX 对象 (ADO) 来检索数据库信息。
  • 理解事务：说明如何在事务环境中运行脚本。商业和信用卡应用程序通常需要具有在事务中运行脚本和组件的能力。
  • 调试 ASP 脚本：讲述如何使用 Microsoft 脚本调试程序来查找和纠正脚本错误。
  • 内建的 ASP 对象：提供对 ASP 内建的对象的快速概述以及指向详细信息的链接。
  • Active Server Pages 对象快速参照卡：
    • 包含关于 ASP 内建的对象的快速参照信息。

本节不包括下列信息：

  • 开发被称为应用程序的可编程的高级脚本集合；请参阅开发 Web 应用程序。
  • 设计 Web 应用程序；请参阅设计决策。
