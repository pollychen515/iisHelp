了解 Active Server Pages

Active Server Pages (ASP) 使生成动态 Web 内容和建立功能强大的 Web 应用变得容易。无论您是 Web 设计人员还是 Web 开发人员，该简介中关于 ASP 的说明都将为您提供帮助。

本节包括：

  • Active Server Pages 简介：对 Active Server Pages 加以概述。
  • ASP 的新增功能：介绍该版本中新添加的功能。
  • ASP 中的重要变化：介绍该版本中有所变化的功能和用法。
  • ASP 教程：为使用 ASP 创建服务器端脚本提供循序渐进的指导课程。

本节不包括下列信息：

  • 有关使用 ASP 编写服务器端脚本的基本概念，请参阅构建 ASP 页。
  • 有关编写更高级脚本的信息，请参阅Web 应用程序：概述。
