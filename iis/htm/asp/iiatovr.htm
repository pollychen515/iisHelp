ASP 指南

Active Server Pages (ASP) 是服务器端设计环境，可用于创建逼真动人的 Web 应用程序。ASP 页是包含服务器端脚本的 HTML 页，Web 服务器先处理这些脚本然后再发送到用户浏览器。不同于以往较难创建的公用网关接口 (CGI) 应用程序，ASP 极大地简化了 Web 应用程序的开发过程。只需几行脚本，就可以将数据库连接或高级自定义功能添加到 Web 页。在以前，必须了解 PERL 或 C 才能添加这些功能，但有了 ASP 就可以使用普通的 Web 脚本语言，例如 Microsoft JScript（符合 ECMA 262 语言规范）、Microsoft Visual Basic VBScript），或者任何与 COM 兼容的脚本语言，包括 JavaScript、PERL 等等。

除了普通的脚本任务，也可以使用 ASP 将脚本扩展到 COM 组件。这些可重复使用的程序模块使您可以将脚本集成为完整的应用程序，以便执行复杂任务，如电子商务的事务处理。

这包含三个模块的指南将逐步介绍一些有用的 Active Server Pages 功能。可以了解到如何使用 VBScript 或 JScript 实现这些功能。

本节包括：

  • 创建 ASP 页：在此模块中，将创建 ASP 页（.asp 文件），并在其中使用一些基本的 HTML 和 Active Server Pages 功能。在开始此模块之前，应该对创建 HTML 页有所了解。
  • 使用 COM 组件：在此模块中，将使用一些 Active Server Pages 附带的 COM 组件。您将了解到如何使用组件旋转 Web 广告横幅、检测浏览器的性能以及在 Web 页中添加数据库功能。在开始此模块之前，最好已有一些使用 JScript 或 Microsoft Visual Basic Scripting Edition (VBScript) 的经验。
  • 编写自己的组件：在此模块中，可以使用 Microsoft Visual Basic 或 Java 来创建自己的财务计算 COM 组件。对于这个模块，必须安装 32 位版本的 Visual Basic 5.0 专业版、Visual Basic 5.0 企业版或更新的版本，或者是 Visual J++ 1.1 或更新的版本，而且您应该有一些 Visual Basic 或 Java 的编程经验。

本节不包含下列信息：

  • 关于编写 ASP 脚本的普通语言信息和主题，请参阅 Active Server Pages。
  • 关于语言参考页，请参阅内置的 ASP 对象。
