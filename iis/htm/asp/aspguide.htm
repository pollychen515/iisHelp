Active Server Pages 指南

欢迎使用 Active Server Pages 指南。本节提供关于如何创建 ASP 网页和开发 Web 应用程序的详细信息。还将讲述可用的与事件相关的方法和接口，它们可用来创建组件供 ASP 或 ISAPI 扩展访问。在本节中，您将找到关于如何存储自定义的 IIS 配置数据以及如何使用内建的对象来管理该数据的信息。最后，指南将提供脚本和程序范例库，用来说明与 IIS 进行交互的各种方式。

本节包括：

  • Active Server Pages：介绍如何创建 Active Server Pages，并解释许多关键概念以便理解 IIS 编程平台。
  • 开发 Web 应用程序：概述关于 IIS 如何适合 Microsoft 分布式应用程序结构的概念，同时提供如何设计和实施应用程序以便通过 HTTP 进行部署的特定详细资料。
  • 程序化管理 IIS：提供概述和参考资料以便帮助您程序化管理 IIS 配置。
  • ASP 内建的对象：讲述 ASP 内建对象的属性和方法。
  • 可安装的 ASP 组件：概述可以安装并能与 ASP 一起工作的各种组件。
  • 脚本参考：包含各种参考资料，包括 @ 指令和 Global.asa 参考。
  • ASP 示例：提供演示脚本以便解释指南中提出的许多概念。
