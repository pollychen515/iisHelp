Active Server Pages

Microsoft Active Server Pages (ASP) 是服务器端脚本环境，可用来创建交互式 Web 页并建立强大的 Web 应用程序。当服务器收到对 ASP 文件的请求时，它处理包含在用于构建发送给浏览器的 Web 页文件中的服务器端脚本。除服务器端脚本外，ASP 文件也可以包含 HTML（包括相关的客户端脚本）和 COM 组件调用，上面谈及的这些组件可执行不同任务，如连接到数据库或处理商业逻辑。

本节中的主题介绍 Active Server Pages，解释 Active Server Pages 脚本的基本概念，并讨论更复杂的应用程序问题，如维护状态。

本节包括：

  • 理解 Active Server Pages：介绍 ASP，讲述本版本的新特性，并包含一个交互式教程。
  • 建立 ASP 页：提供编写脚本所需的基本信息，并介绍从基本语法到创建事务脚本的一系列主题。

本节不包含下列信息：

  • 关于更多高级主题，如设计和建立 Web 应用程序，请参阅 Web 应用程序：概述。

  • ASP 附带组件的参考页；请参阅 可安装的 ASP 组件。
  • 示例脚本；请参阅 ASP 示例。
