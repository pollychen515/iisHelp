对 Internet 服务管理器 (HTML) 的访问仅限于 Localhost
作为安全预防措施，安装程序将限制 Internet 服务管理器 (HTML) 以便仅允许从服务器计算机自身（Localhost，IP 地址 127.0.0.1）执行访问。

请尝试下列操作：

要允许远程使用 Internet 服务管理器 (HTML) 来管理该服务器，请转到服务器并使用 IIS 管理单元来更新对 Web 站点 IISADMIN 的 IP 地址限制。

要点 只有当服务器在 Windows 2000 Server 或 Windows 2000 Advanced Server 平台上运行 IIS 时，才可以修改 IP 地址限制。不能在 Windows 2000 Professional 平台上使用 IIS 来修改 IP 地址限制。

在 Internet 信息服务管理单元中，

  1. 展开您目前连接的 Web 站点。
  2. 选择 IISADMIN 虚拟目录。
  3. 在“动作”菜单中，选择“属性”。
  4. 选择“目录安全性”选项卡。
  5. 编辑 IP 地址和域名限制。
